# How

    Usage:
      thor kuroon:clone --repo=REPO

    Options:
      --repo=REPO
      [--from=FROM]
                                   # Default: bitbucket
      [--to=TO]
                                   # Default: github
      [--private], [--no-private]

    clone a repo (dunyakirkali/project) from BitBucket or Git to BitBucket or Git
